Étape 1 : Trouver un hébergeur

Un hébergement sur le site 000webhost

  1. Créer un compte sur le site Hébergement Web Gratuit
  2. Valider le reCAPTCHA :
  3. Consulter votre boîte mail :
  4. Vous devez normalement recevoir un message dans votre boîte mail, cliquez dessus :
  5. Cliquez sur le bouton "Vérifier l'e-mail" :
  6. Vous devriez être rediriger vers cette page
  7. Choisissez "Créer une conception Web"
  8. Poursuivez en cliquant sur "Autre"
  9. Vous pouvez passer cette étape :
  10. Choisissez le nom de votre site web et le mot de passe puis "Soumettre" :
  11. Sélectionnez "Upload your site" :
  12. Survolez cette image et cliquez sur "Gérer le site" :
  13. Vous arrivez sur le dashboard :
  14. Sur le sidebar, cliquez sur "Paramètre du Site Web" puis sur "Général" :
  15. Noter vos informations FTP, le mot de passe est celle que vous aviez choissit lors de la création du nom du site web :
  16. Installer Filezilla en le téléchargeant ici : FileZilla
  17. Créer un nouveau site dans le Gestionnaire de Site de Filezilla en cliquant sur l'icône et non la flèche, vous verrez un bouton "Nouveau Site", cliquez dessus et donner un nom à votre site web :
  18. Paramètrez votre connexion FTP :
  19. Cliquez sur OK pour terminer
  20. Connectez-vous sur votre site en cliquant sur la flèche bas :
  21. Si tout se passe bien, vous devriez avoir ce résultat :
  22. Créer un fichier index.html :
  23. Ajouter ce code dans le fichier index.html et pensez à enregistrer : 
    <!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <title>Page web</title>
</head>
<body>
    <h1>Titre du site</h1>
</body>
</html>
